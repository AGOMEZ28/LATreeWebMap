Los Angeles Tree Web Map

Filter Buttons

The filter buttons are located on the top-left corner and are used to seach between the Los Angeles park tree database or the Los Angeles street tree database

Another set of filter buttons are displayed to perform a search by street name, park name, or tree name


A search bar used to type in a search

Basemap Options

Explore the various basemaps located on the top-right corner and select the one best suited for your needs

Symbology Buttons

The symbology buttons are located on the bottom-left corner and are used to change the map symbols based on the properties

Legend

The legend is located on the bottom-right corner and updates according to the symbology that is currently active

Zoom/Home

The +/- buttons located on the top left are used to zoom in and out of the map


The home button is used to return to the default map extent

The Los Angeles Tree Web Map is a mapping application created by Alejandro Gomez for the University of Wisconsin-Madison Online GIS Development Master's Program practicum project. This application is used to explore the City of Los Angeles tree database at public parks and public street right-of-way (ROW). The map was produced using Esri Leaflet API in combination with Bootstrap, JQuery, and AJAX.

All data displayed on the map is from the Los Angeles Geohub
