Los Angeles Tree Web Map

Filter Buttons

These filter button are located on the top-left corner and are used to seach between the Los Angeles park tree database or the Los Angeles street tree database.

Basemap Options

Explore the various basemaps located on the top-right corner and find the one that is most suited for your needs.

Symbology Buttons

The symbology buttons are located on the bottom-left corner and are used to change the map symbols based on the properties.

Legend

The legend is located on the bottom-right corner and updates based on what symbology button is active.

Chart

The chart displays the current zoom data in bar graph format for easy visualization comparisons.

The Los Angeles Tree Web Map is a mapping application created by Alejandro Gomez for the University of Wisconsin-Madison GIS Development Master's Program Practicum project. This application is used to explore the City of Los Angeles tree database at public parks and public street right-of-way (ROW). The map was produced using Esri Leaflet API in combination with Bootstrap, JQuery, and AJAX.

All data displayed on the map is from the Los Angeles Geohub
